Projetos em Ciência de Dados

Construção de um programa de fidelidade com clusterização de clientes

Usei Python, Estatística e técnicas não-supervisionadas de Machine Learning para segmentar um grupo de clientes com base em suas características de performance de compra, a fim de, selecionar grupos de clientes para formar um programa de fidelidade com o objetivo de aumentar a receita da empresa. O resultado dessa solução, caso implementada, seria de R$ 15MM de dólares de receita anual.

Ferramentas utilizadas:

  • Jupyter Notebook;
  • K-Means, Herarquical Clustering, DBScan;
  • AWS Cloud (EC2, S3, Postgres, SQLite);
  • Metabase Visualization.
  • SAIBA MAIS
